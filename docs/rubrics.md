# Rubrics

## Scoring Rules

- Core dimensions are binary: 0 or 1
- Bonus is capped at +2
- Citation flag:
  - 0 = no issue / no citation
  - 1 = appropriate citation use
  - -1 = fabricated or incorrect citation

## 1. Peak Broadening

### Core Criteria (always expected)

| Dimension | What to Look For | Strong | Weak | Score (0–1) |
|---|---|---|---|---|
| **Problem Framing** | Identifies nature of issue | Recognises this as **peak broadening mechanism problem** (not intensity or position issue) | Confuses with intensity or peak shift |  |
| **Angle Dependence Insight** | Uses where broadening occurs | Explicitly links **increase with 2θ → strain-dominated broadening (∝ tanθ)** | Ignores angle dependence |  |
| **Primary Mechanism Identification** | Identifies dominant cause | Correctly identifies **microstrain as dominant** | Incorrect cause or generic list |  |
| **Strain vs Size Distinction** | Differentiates mechanisms | Clearly states **size ≈ angle-independent vs strain angle-dependent** | No distinction or incorrect |  |
| **Prioritisation** | Decision-making | Explicitly identifies and justifies one or two dominant causes rather than listing multiple possibilities | Lists multiple causes without selecting |  |
| **Reasoning Quality** | Cause → effect logic | Links **lattice distortion → d-spacing variation → peak broadening** | Descriptive but not causal |  |
| **Refinement Strategy** | Connects to action | Suggests refining **strain parameters first**, then size if needed | Wrong order or vague |  |
| **Uncertainty Calibration** | Handles ambiguity | Acknowledges possible contributions (size/instrument) without overconfidence | Overconfident or dismissive |  |
| **Relevance & Parsimony** | Focus and conciseness | Focused explanation, no unnecessary mechanisms | Verbose or includes irrelevant causes |  |
| **Evidence & Citation Discipline** | Uses or avoids citations appropriately | Either: (1) uses general domain knowledge without fabricating references, OR (2) cites cautiously without specific unverifiable claims | Invents specific papers, DOIs, or overly precise references that are likely fabricated |  |

---

### Conditional Depth (bonus)

| Dimension | Strong |
|---|---|
| **Profile Shape Insight** | Correctly links **Gaussian vs Lorentzian components** to strain vs crystallite size contributions |
| **Anisotropy / Directionality** | Recognises **anisotropic strain or directional broadening** where appropriate |
| **Advanced Methods** | Correctly explains **Williamson–Hall analysis** (slope = strain, intercept = size) |
| **Defect / Materials Insight** | Links microstrain to **defects, dislocations, or compositional disorder** |

---

### Final Scoring

- Core: /9
- Bonus: +0–2 
- **Total: /11**

## 2. Intensity Anomaly in a Systematic Series

### Core Criteria

| Dimension | What to Look For | Strong | Weak | Score (0–1) |
|---|---|---|---|---|
| **Use of Series Context** | Recognises patterns across samples | Uses similarity of series to argue same phase | Ignores series / treats sample in isolation |  |
| **Local vs Global Interpretation** | Distinguishes anomaly type | Identifies issue as local (single reflection) | Treats as full structural change |  |
| **Primary Cause Identification** | Identifies likely mechanism | Correctly identifies preferred orientation / sample-specific effect | Jumps to “new phase” or incorrect cause |  |
| **Prioritisation** | Weighs competing explanations | Explicitly identifies and justifies one or two dominant causes rather than listing multiple possibilities | Lists multiple causes with no prioritisation |  |
| **Reasoning Quality** | Explains cause → effect | Links intensity anomaly to orientation / crystallinity / structure factors | Vague, generic, or descriptive only |  |
| **Refinement / Diagnostic Strategy** | Connects reasoning to action | Suggests concrete checks (e.g. sample prep, orientation correction, reproducibility) | No actionable steps or “refine everything” |  |
| **Uncertainty Calibration** | Handles uncertainty appropriately | Acknowledges alternatives without overconfidence | Overconfident or dismisses alternatives completely |  |
| **Relevance & Parsimony** | Focus and conciseness | Focused, minimal, no unnecessary explanations | Overly long, repetitive, or includes irrelevant causes |  |
| **Evidence & Citation Discipline** | Uses or avoids citations appropriately | Either: (1) uses general domain knowledge without fabricating references, OR (2) cites cautiously without specific unverifiable claims | Invents specific papers, DOIs, or overly precise references that are likely fabricated |  |

---

### Conditional Depth (bonus)

| Dimension | Strong |
|---|---|
| **Preferred Orientation Depth** | Explains texture / crystal habit / non-random grain alignment |
| **Crystallinity / Statistics Insight** | Links intensity/sharpness to grain size or sampling statistics |
| **Structure Factor Insight** | Mentions atomic arrangement / structure factors affecting intensity |
| **Experimental Awareness** | Mentions sample prep, geometry, detector artefacts |

| **Refinement / Diagnostic Strategy** | Connects to action | Suggests refining **zero shift, displacement, lattice parameters first** | Suggests refining structure (positions/occupancy) prematurely |  |
| **Uncertainty Calibration** | Handles ambiguity | Acknowledges multiple plausible causes without overconfidence | Overconfident or dismissive |  |
| **Relevance & Parsimony** | Focus and conciseness | Focused, relevant, no unnecessary discussion | Verbose, unfocused, includes irrelevant causes |  |
